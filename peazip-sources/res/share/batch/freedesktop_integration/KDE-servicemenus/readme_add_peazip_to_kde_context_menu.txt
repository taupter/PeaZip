TO ADD PEAZIP TO KDE6 CONTEXT MENU

1) Copy .desktop files from KDE6-dolphin subfolder to (usually):
/usr/share/kio/servicemenus (all users)
~/.local/share/kio/servicemenus (current user only)

The .desktop files from KDE5-dolphin subfolder currently works too with KDE6.
Please note that "Allow executing file as program" or "Is executable" permission may be needed to be set for the .desktop files in context menu.
